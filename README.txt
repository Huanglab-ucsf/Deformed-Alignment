These source codes are distributed as accompanying codes for the article "Deformed Alignment of Super-Resolution Images for Semi-flexible Structures in 3D" by Bo Huang group.

The provided package includes four model-free fast Matlab softwares for 
(1) 2D reformative alignment of super-resolution images; 
(2) 3D alignment; 
(3) 2-color 2D alignment; 
(4) 2-color 3D alignment.  

Each software is distributed with examples used in the article.
